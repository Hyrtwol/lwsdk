8.3 Changes

January 31st, 2005

This is a list of the changes in the LightWave 8.3 release that affect the SDK.

Miscellaneous

  • A maxPointsPerPoly value has been added to the LWObjectFuncs structure in the lwmeshes.h header. This value lets a plug-in determine the maximum number of points that can be used to construct a polygon in the current application.

  • A new event, DTEVENT_BAKEZONECHANGE, has been added to the Dopetrack Proxy global to notify clients when the user modifies the boundaries of defined baking zones.
