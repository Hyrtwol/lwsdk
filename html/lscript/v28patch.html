 Bug Fixes 


 Behavioral Changes  

                                                                                           
  The numeric value returned by the Scene Object Agent's alertlevel data member was
  improperly zero-based (ranging from 0-2 in the current release) instead of LScript's
  one-based paradigm.  This caused issues with the AlertLevel() command who was expecting
  a one-based argument value.
