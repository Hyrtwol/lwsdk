9.3.1 Changes

October 11, 2007

This is a list of the changes in the LightWave® 9.3.1 release that affect the SDK.

lwmaster.h

  • Added master plugin handler notifications:
    • LWEVNT_NOTIFY_TEXTURE_REMOVING
    • LWEVNT_NOTIFY_TEXTURE_ALTERED
    • LWEVNT_NOTIFY_IMAGE_REMOVING
    • LWEVNT_NOTIFY_IMAGE_ALTERED

Commands: Layout

  • New commands added:
    • BlurBackgroundRadiosity
    • SelectByPartialName
  • Behaviors changed:
    • SelectByName no longer also performs partial matching. If you need partial text matching for selections, please now use SelectByPartialName

lwrender.h

  • A new state value, loadInProgress, has been added to the Scene Info Global. This setting represents the state of scene loading within the application.

LScript v2.12

  • Added the REALISTIC fog type value to the LScript environment.
